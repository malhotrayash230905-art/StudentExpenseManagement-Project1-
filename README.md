# CampusBudget Buddy - Student Expense Management System

A sleek, modern, and interactive expense tracking application designed for students. Built with **HTML**, **Vanilla CSS**, and **Supabase** for persistent data storage.

### 🔗 [Live Demo - Try it out!](https://student-expense-management-project1.vercel.app/)

## 🚀 Features
- **Smart Budgeting**: Set monthly budgets and track spending in real-time.
- **Expense Categorization**: Group spending into categories (Food, Travel, etc.) for better insights.
- **Glassmorphism UI**: High-end modern design with vibrant colors and smooth animations.
- **Persistent Storage**: All data is securely stored and synced using Supabase.
- **Interactive Visuals**: Clean, responsive dashboard with intuitive expense management.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla CSS3, Javascript (ES6+)
- **Backend/Database**: Supabase
- **Icons/Graphics**: Lucide-style iconography and custom illustrations

## 📦 Setup & Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/student-expense-management.git
   ```
2. Open `index.html` in your browser.
3. (Optional) Replace the Supabase credentials in `supabaseClient.js` with your own project details to host your own version.

## 🔐 Security Note
This project uses a Supabase **Anon Key**, which is safe to be exposed in the frontend as it respects Row-Level Security (RLS) policies configured on the backend. For production use, ensure your Supabase RLS is properly set up.
